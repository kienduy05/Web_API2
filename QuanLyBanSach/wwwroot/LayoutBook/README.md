# BookSelling - Online Book Store Website

**BookSelling** is a fully responsive **Online Book Store HTML template**.  
It features a clean, modern, and elegant UI for showcasing and selling books online.  
Designed and developed by **Devomate**, this template includes all essential pages required for a complete book-selling website.

---

## 🌐 Pages Included

- Home  
- Shop  
- Product Detail Page  
- Cart  
- Wishlist  
- About Us  
- Contact Us  
- Login  
- Register  

---

## ⚡ Features

- Fully responsive layout (desktop, tablet, mobile)  
- Built using **HTML, CSS, JS, and Bootstrap 5**  
- Modern **book store UI design**  
- Filter options and category sections  
- Wishlist and cart UI  
- Product detail page with price, description, and “Add to Cart”  
- Easy to customize  
- Cross-browser compatible
